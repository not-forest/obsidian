> To say about the nationality, we connect the **nation** word with + **사람** or **인**
> - 폴란드사람;
> - 폴란드인;
> To create the name of the **language** we connect the **nation** word with + **어**
> - 한극어;
> There is an **exception** for english language: **영어**

# Structure (noun + 여요/이여요)

To create a structure of type "*this is a noun*" we add **여요 or 이여요** to the end of the noun. The first case is used when **noun ends on a vowel**. Otherwise the second, longer part is used and added.

| cook    | 요리사  |
| ------- | ---- |
| doctor  | 의사   |
| player  | 운동선수 |
| teacher | 선생님  |
| singer  | 가수   |
| actor   | 배우   |
### Exercise 1
1. 요리나여요 - He/She is a cook.
2. 운동선수여요 - He/She is a player.
3. 선생님이여요 - He/She is a teacher.
4. 가수여요 - He/She is a singer
5. 의사여요 - He/She is a doctor
6. 배우여요 - He/She is an actor

# Structure (noun + 은/는)

Used when we want to *mark* the topic of the sentence. It can be understood as a combination of words: am, is, are. The second one is used with **vowels**. Otherwise the first one is used.

저극 선생님이여요 - I am a teacher
제훈은 요리사여요 - Dziechun is a cook
